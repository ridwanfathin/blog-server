# echo-server

## Description 
Simple Sample of web serverusing Echo Golang Framework.

## Installation
Clone the project:
```
  git clone https://github.com/ridwanfathin/echo-server
 ```
 open the directory:
 ```
  cd echo-server
 ```
 
 define `GOPATH`:
 ```
  export GOPATH=[your project path]
 ```
 
 build binaries: 
 ``` 
 go install main
 
 ```
 
 run the server: 
 
 ```
 bin/main
 
 ```
