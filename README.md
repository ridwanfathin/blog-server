# blog-server

## Description 
Restful API Web server for simple blog using Echo framework, Gorm, and Postgresql. 

## Installation
Clone the project:
```
  git clone https://github.com/ridwanfathin/blog-server
 ```
 open the directory:
 ```
  cd echo-server
 ```
 
 define `GOPATH`:
 ```
  export GOPATH=[your project path]
 ```
 
 build binaries: 
 ``` 
 go build main
 
 ```
 
 run the server: 
 
 ```
 ./main (linux & Mac)
 main.exe (windows)

 
 ```
